# T-M2-3: 動画ダウンロード・メタデータ保存の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M2 |
| 依存 | T-M2-2 |
| 要件ID | FR-008, FR-009, FR-010 |

## 説明

生成された動画をダウンロードし、`dist/` ディレクトリに安全に保存する処理を実装する。`.part` 拡張子による一時保存とリネーム方式で破損を防止し、メタデータ JSON も並行して保存する。

## 作業内容

1. `src/downloader.ts` を作成する
2. `dist/` ディレクトリの存在確認・自動作成処理を実装する
3. ファイル名を生成する処理を実装する（`{YYYYMMDDHHmmss}_{6文字ランダム}.mp4`）
4. `ai.files.download()` で動画をダウンロードする処理を実装する
   - ダウンロード先は `.part` 拡張子で一時保存する
   - ダウンロード完了後に `.mp4` にリネームする
   - ダウンロード失敗時は `.part` ファイルを削除する
5. メタデータ JSON を生成・保存する処理を実装する
   - 動画ファイルと同じベースネームで `.json` として保存する
   - prompt, model, resolution, aspectRatio, duration, createdAt, outputFile を含む

## タスクのチェックリスト

- [ ] dist/ ディレクトリが自動作成される
- [ ] ファイル名が {タイムスタンプ}_{ランダム}.mp4 形式になる
- [ ] ダウンロード中は .part 拡張子で保存される
- [ ] ダウンロード完了後に .mp4 にリネームされる
- [ ] メタデータ JSON が動画と同名で保存される
- [ ] JSON にプロンプト・設定値・日時が含まれている
