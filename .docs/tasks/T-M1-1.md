## 説明

Node.js + TypeScript プロジェクトの基盤を構築する。パッケージマネージャの初期化、TypeScript 設定、ビルドツール設定、環境変数テンプレートの作成を行う。

## 作業内容

1. `package.json` を作成し、`type: "module"` を設定する
2. 依存パッケージをインストールする
   - `@google/genai`, `commander`, `dotenv`（dependencies）
   - `typescript`, `tsup`, `@types/node`（devDependencies）
3. `tsconfig.json` を作成し、strict モード・ESM 設定を行う
4. `tsup.config.ts` を作成し、TypeScript ビルド設定を行う
5. `package.json` に npm scripts を追加する
   - `build`: tsup でビルド
   - `generate`: ビルド成果物を実行して動画生成
6. `.env.example` を作成する（`GOOGLE_API_KEY=your_api_key_here`）
7. `.gitignore` を更新し、以下を除外対象に追加する
   - `.env`, `.env.*`（`.env.example` は除外しない）
   - `dist/`
   - `*.part`
8. `src/` ディレクトリを作成する

## タスクのチェックリスト

- [ ] `npm install` が成功する
- [ ] `npm run build` が成功する
- [ ] `.env.example` が存在する
- [ ] `.gitignore` で `.env`, `dist/`, `*.part` が除外されている
- [ ] `src/` ディレクトリが存在する
