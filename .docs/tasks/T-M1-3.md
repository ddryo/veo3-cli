## 説明

プロジェクト全体で使用する TypeScript 型定義を作成する。CLI オプション、設定、メタデータなどの型を定義する。

## 作業内容

1. `src/types.ts` を作成する
2. CLI オプションの型を定義する（model, resolution, aspect, duration）
3. 解決済み設定の型を定義する（apiKey, model, modelAlias, resolution, aspectRatio, duration）
4. メタデータの型を定義する（prompt, model, resolution, aspectRatio, duration, createdAt, outputFile）
5. モデル・解像度・アスペクト比・尺の許容値をリテラル型で定義する

## タスクのチェックリスト

- [x] 全ての型が定義されている
- [x] 許容値がリテラル型で制約されている
- [x] ビルドが型エラーなく通る
