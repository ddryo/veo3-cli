# T-M3-1: CLI エントリポイントの実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M3 |
| 依存 | T-M2-1, T-M2-3 |
| 要件ID | FR-001, FR-002, FR-003, FR-004, FR-005 |

## 説明

commander を使用して CLI のエントリポイントを実装する。プロンプトの受け取り、全オプションの定義、各モジュールの呼び出しフローを統合する。

## 作業内容

1. `src/cli.ts` を作成する
2. commander でプログラムを定義する
   - プロンプトを必須の引数として受け取る
   - `--model`（-m）: fast / standard、デフォルト fast
   - `--resolution`（-r）: 720p / 1080p / 4K、デフォルト 720p
   - `--aspect`（-a）: 16:9 / 9:16、デフォルト 16:9
   - `--duration`（-d）: 4 / 6 / 8、デフォルト 8
3. 実行フローを統合する
   - config.ts で設定を解決する
   - validator.ts でバリデーションする
   - generator.ts で動画を生成する
   - downloader.ts でダウンロード・保存する
4. `package.json` の `generate` スクリプトを設定する

## タスクのチェックリスト

- [ ] `npm run generate -- "プロンプト"` で実行できる
- [ ] 全オプション（--model, --resolution, --aspect, --duration）が動作する
- [ ] プロンプト未指定時にヘルプが表示される
- [ ] 設定解決 → バリデーション → 生成 → ダウンロードの順で処理が実行される
