# T-M2-1: パラメータバリデーションの実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M2 |
| 依存 | T-M1-2, T-M1-3 |
| 要件ID | FR-006 |

## 説明

ユーザーが指定したパラメータの組み合わせが有効かどうかを検証するバリデーションモジュールを実装する。無効な組み合わせを動画生成 API 呼び出し前に検出し、明確なエラーメッセージを返す。

## 作業内容

1. `src/validator.ts` を作成する
2. 解像度と尺の組み合わせバリデーションを実装する
   - 720p: 4, 6, 8 秒 OK
   - 1080p: 8 秒のみ OK
   - 4K: 8 秒のみ OK
3. 各パラメータの許容値チェックを実装する（resolution, aspect, duration, model）
4. バリデーションエラー時に、どのパラメータが不正でどの値が許容されるかを示すメッセージを返す

## タスクのチェックリスト

- [ ] 1080p + 4秒 の組み合わせがエラーになる
- [ ] 4K + 6秒 の組み合わせがエラーになる
- [ ] 720p + 4秒 の組み合わせが成功する
- [ ] 不正な解像度値（例: 480p）がエラーになる
- [ ] エラーメッセージに許容される値が含まれている
