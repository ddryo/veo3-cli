# T-M2-2: 動画生成・ポーリング処理の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M2 |
| 依存 | T-M1-2, T-M1-3 |
| 要件ID | FR-001, FR-007, NFR-005 |

## 説明

`@google/genai` SDK を使用して Veo 3 API で動画生成を開始し、ロングランニングオペレーションのポーリングで完了を待機する処理を実装する。ポーリング中は経過時間を CLI に表示する。

## 作業内容

1. `src/generator.ts` を作成する
2. `@google/genai` の `GoogleGenAI` クラスをインスタンス化する処理を実装する
3. `ai.models.generateVideos()` で動画生成を開始する処理を実装する
   - model, prompt, config（resolution, aspectRatio）を渡す
4. ポーリングループを実装する
   - `ai.operations.getVideosOperation()` で状態を確認する
   - 10 秒間隔でポーリングする
   - 経過時間をコンソールに表示する
5. タイムアウト処理を実装する（最大 6 分）
6. オペレーションの失敗終端（failed / cancelled 等）を即時検出し、理由付きでエラーを返す
7. 完了時に生成された動画情報を返す
